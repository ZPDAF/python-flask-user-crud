# 运行步骤
0. 安装python3、mysql5.7
1. 安装模块
```bash
pip install flask
pip install pymysql
```
2. 执行Mysql脚本： `user_test.sql`
3. 项目根目录，命令行执行
```bash
python app.py
```
4. 访问系统：
`localhost:8001`
